## Markdown editor

支持编辑预览，具有本地保存功能，支持关闭页面后下次继续编辑。考虑到有可能需要打印成pdf文件，所以添加了打印预览功能，可以利用浏览器自身的打印功能导出pdf文件。

点击 [这儿](http://wy-ei.github.io/markdown-editor) 尝试使用。
